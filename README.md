# Assignment-and_Summer_Fes

# 要件定義書

## 1. プロジェクト名
UsefulTools Hub (便利ツール集約サイト)

## 2. 背景・目的
本プロジェクトは、学校の課題および部活動の制作物として作成されるものであり、Djangoフレームワークを使用して「日常的に使える便利なツールを集約したWebアプリケーション」を構築することを目的とする。Pythonのライブラリを3つ以上使用することが課題要件であり、開発を通じてフルスタック開発の基礎を実践する。学生や一般ユーザーが頻繁に使用する実用的なツールを一箇所にまとめることで、利便性の高いサービスを提供する。

## 3. 想定ユーザー
- 学内の教員や生徒（評価用・日常使用）  
- 開発者本人（実装と発表）
- 一般ユーザー（学生、社会人、クリエイター等）
- デザイナー・開発者（色彩ツール、QRコード生成等）

## 4. 機能要件

### 4.1 画像結合機能
- 複数枚の画像をユーザーがアップロード可能
- 横方向または縦方向に画像を結合して1枚にする
- 結合後の画像をダウンロード可能にする

### 4.2 ユーザーレコメンド・行動分析機能
- 各機能（ツール）の利用履歴をユーザー単位で記録（閲覧ログ、タグ付け等）
- ログに基づいて類似した傾向の記事やツールを推薦（TF-IDF＋類似度検索）
- タグの出現頻度から「おすすめタグ」や「人気ツール」を表示
- Pythonライブラリ `scikit-learn` を使用してレコメンドを実装
- ユーザー行動データを円グラフや棒グラフで可視化（`matplotlib` + `numpy`）
- 将来的に自動クラスタリングや傾向分析も検討（KMeans等）

### 4.3 QRコード生成機能
- テキスト、URL、連絡先情報等をQRコードに変換
- 生成したQRコードの画像ダウンロード機能
- サイズ調整、エラー訂正レベル設定機能

### 4.4 パスワード生成機能
- セキュアなランダムパスワードの生成
- 文字数、文字種（大文字、小文字、数字、記号）のカスタマイズ
- 生成履歴の一時保存機能（セキュリティ考慮）

### 4.5 色彩ツール機能
- カラーパレット生成（補色、類似色、トライアド等）
- HEX、RGB、HSL形式の相互変換
- 色覚異常シミュレーション機能

### 4.6 テキスト変換機能
- 文字数カウント、改行除去、大文字小文字変換
- URLエンコード/デコード
- Base64エンコード/デコード
- ひらがな・カタカナ変換

### 4.7 ユニット変換機能
- 長さ、重さ、温度、通貨等の単位変換
- 外部APIを利用したリアルタイム為替レート取得
- よく使用する変換の履歴保存

### 4.8 データ可視化機能
- CSVファイルアップロードによるグラフ自動生成
- `matplotlib`を使用した棒グラフ、折れ線グラフ、円グラフ作成
- `numpy`による統計情報表示（平均、標準偏差等）
- ユーザーのツール利用傾向、投稿内容のキーワード出現頻度なども可視化対象に追加

### 4.9 Webスクレイピング機能
- `beautifulsoup4`と`requests`を使用したWebページ情報抽出
- 価格比較、ニュース取得、天気情報収集
- 取得結果のCSV出力機能

### 4.12 リアルタイム通信機能
- `socket`を使用したチャット機能
- ファイル共有、メッセージ履歴保存
- 複数ユーザー対応（`threading`活用）

### 4.13 感想・質問コーナー機能
- 各ツールに対してユーザーが感想や質問を自由に投稿できる機能
- 投稿はユーザーごとに記録され、管理画面で一覧・削除可能（Django Admin）
- 投稿された内容を集計・分析し、「よくある質問」や「人気の意見」を抽出
- 感想内容を形態素解析して、キーワードランキングを表示（`collections.Counter`等）
- 将来的に感情分析（ポジ/ネガ）なども検討可能（テキスト解析拡張）

| 分類       | 技術名           | 用途                     |
|------------|------------------|--------------------------|

| ライブラリ   | beautifulsoup4   | Webスクレイピング・HTML解析 |

| ライブラリ   | matplotlib       | データ可視化・グラフ生成    |
| ライブラリ   | numpy            | 数値計算・統計処理         |
| ライブラリ   | qrcode           | QRコード生成              |
| ライブラリ   | colorsys         | 色彩変換・計算            |
| ライブラリ   | secrets          | セキュアなパスワード生成    |
| ライブラリ   | socket           | リアルタイム通信           |
| ライブラリ   | threading        | マルチスレッド処理         |
| ライブラリ   | scikit-learn     | レコメンド・行動分析       |
| その他     | HTML/CSS/JavaScript | フロントエンド UI        |

## 6. 非機能要件
- システムはローカルホスト上で動作（開発段階）
- Django管理画面によるDB確認が可能
- ユーザー入力に対して3秒以内にレスポンスを返すことを目標
- 簡易的なUIを提供（Bootstrap等のCSSフレームワークの使用可）

## 9. 期待される効果・アピールポイント
- **実用性**: 日常的に使える機能を集約し、ユーザーの利便性を向上
- **技術的多様性**: 画像処理、AI解析、機械学習、リアルタイム通信等の最新技術活用
- **データサイエンス**: matplotlib・numpy・scikit-learnによる本格的なデータ分析

- **拡張性**: 15以上のPythonライブラリを統合した高度なアーキテクチャ

- **可視化分析強化**: 投稿分析や人気タグの可視化、個人ダッシュボードの生成
- **感想投稿分析**: よくある質問や人気の意見を抽出し、AI風に提示
